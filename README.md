## 📂 Project Structure & Architecture

MN ZONEX 2.0 follows a clean, modular architecture optimized for speed and local-first execution.

```text
MN-ZONEX-2.0/
├── index.html              # Main Landing Page (Hero, Direct Services, Overview)
├── business.html           # Business Hub (Partners: KILLERS VIP, ZERO CORE & Solutions)
├── services.html           # Service Directory (Web Dev Packages, Creative Studio, External)
├── games.html              # Gaming Universe (Neuro-Link, Circle Dash, Dino, etc.)
├── tools.html              # Smart Tools Hub (Direct access to 50+ utilities)
├── social.html             # Community & Social Connect Hub
├── about.html              # Brand Story & Mission
├── feedback.html           # User Community Voice & Reviews
├── assets/                 # Core System Assets
│   ├── css/                # Base, Layout, Components, Animations, Responsive
│   ├── js/                 # main.js (System Logic), Particles, Scroll Effects
│   └── images/             # Professional content & branding assets
├── tools/                  # 50+ Dedicated HTML/JS Utility Tools
├── games/                  # Source folders for individual high-speed games
└── config/                 # theme.js (Global styling & Feature toggles)
```
